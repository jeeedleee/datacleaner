# datacleaner 安装说明

## 文件清单

本安装包包含以下文件：

- `datacleaner_0.1.0.tar.gz` - R 包源码包
- `install.R` - 一键安装脚本
- `安装说明.txt` - 本文件

## 安装方法

### 方法一：使用一键安装脚本（推荐）

1. 将所有文件放在同一个文件夹中
2. 打开 RStudio
3. 在 R 控制台中运行：

```r
setwd("你的文件夹路径")  # 例如：setwd("C:/Users/用户名/Downloads")
source("install.R")
```

脚本会自动：
- 配置国内镜像
- 检查并安装依赖包
- 安装 datacleaner
- 验证安装

### 方法二：手动安装

如果方法一失败，可以手动安装：

1. 先安装依赖包：

```r
install.packages(c("shiny", "miniUI", "DT", "httr2",
                   "readr", "readxl", "rstudioapi",
                   "jsonlite", "yaml"))
```

2. 安装 datacleaner：

```r
install.packages("datacleaner_0.1.0.tar.gz", repos = NULL, type = "source")
```

## 首次使用

安装完成后：

1. 在 RStudio 中，点击菜单：**Addins** → **数据清洗助手**

2. 在打开的界面中，点击 **设置** 标签页

3. 填写配置信息：
   - **API 密钥**：你的 API 密钥
   - **模型**：模型名称（如 gpt-4）

4. 点击 **保存设置**

配置会保存在 `~/.datacleaner/config.yml`，下次使用时自动加载。

## 常见问题

**Q: 安装时提示缺少依赖包？**
A: 运行 install.R 脚本会自动安装依赖包。如果失败，请手动安装。

**Q: 提示需要 Rtools？**
A: 源码包安装可能需要 Rtools。如果遇到问题，请安装 Rtools：
   https://cran.r-project.org/bin/windows/Rtools/

**Q: API 地址是什么？**
A: API 地址已固定为 https://537-ai.net，无需配置。

**Q: 如何更新到新版本？**
A: 重新运行安装脚本即可。

## 使用流程

1. 点击 Addins → 数据清洗助手
2. 导入数据文件（CSV 或 Excel）
3. 在清洗规则标签页指定规则
4. 点击生成代码
5. 代码会自动插入到 RStudio 编辑器
